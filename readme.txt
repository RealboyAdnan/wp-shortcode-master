== Description ==
WP Shortcode Master is a WordPress plugin that allows you to manage custom shortcodes using a custom post type. It also provides a shortcode inserter button in the WordPress editor for easy insertion of shortcodes into your content.

== Features ==
- Register a custom post type for managing shortcodes
- Display shortcode column in the shortcode custom post type listing
- Customize the admin columns for shortcode custom post type
- Add shortcode inserter button in the WordPress editor
- Search and select shortcodes to insert into content

== Installation ==
1. Upload the `wp-shortcode-master` folder to the `/wp-content/plugins/` directory.
2. Activate the WP Shortcode Master plugin through the 'Plugins' menu in WordPress.

== Usage ==
1. After activating the plugin, you will find a new menu item called "ShortCodes" in the WordPress admin.
2. Click on "ShortCodes" to manage your custom shortcodes.
3. To insert a shortcode into your content, click the "Shortcode" button in the WordPress editor toolbar.
4. A shortcode selection dialog will appear. You can search for shortcodes and select the one you want to insert.
5. Click the "Insert Code" button to insert the selected shortcode into your content.
